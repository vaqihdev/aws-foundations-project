# Amazon EC2 Instance

## Overview

Amazon Elastic Compute Cloud (EC2) provides scalable virtual servers in the AWS Cloud. In this project, an EC2 instance is deployed within the public subnet to host an application server.

## CloudFormation Resource

```yaml
AppInstance:
  Type: AWS::EC2::Instance
```

## Configuration

```yaml
InstanceType: t3.micro
ImageId: !Ref AmazonLinuxAMIID
SubnetId: !Ref PublicSubnet
SecurityGroupIds:
  - !Ref AppSecurityGroup
```

## Instance Details

| Setting          | Value              |
| ---------------- | ------------------ |
| Instance Type    | t3.micro           |
| Operating System | Amazon Linux 2     |
| Network          | Public Subnet      |
| Security         | App Security Group |

## Why This Configuration?

The EC2 instance is deployed in the public subnet so that it can receive incoming web traffic. Amazon Linux 2 provides a lightweight and secure operating system commonly used for AWS workloads.

The t3.micro instance type was selected because it is cost-effective and suitable for learning environments and small workloads.
